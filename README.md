# AI 内容工厂

帮大型连锁店店员快速生成合规社媒内容的 AI 工具。48h MVP。

## 解决什么问题

药店需要持续产出小红书、朋友圈等营销内容，
但店员没有内容创作能力，医药类文案又有严格的合规红线。

## 核心流程

拍产品照片 → AI 识别商品 → 生成营销文案 → 合规检测 → 一键复制发布

四步闭环，店员不需要任何写作能力。

## 设计决策

从三类角色的实际使用场景反推产品逻辑：
- 门店店员：操作要快，不能有学习成本
- 新媒体运营：需要风格可控、内容多样
- 甲方管理层：必须规避合规风险

Prompt 架构分 6 层：
全局合规层 / 品类知识包 / 内容模板 / 爆文参考 / 产品输入 / 合规边界

每一层独立，便于后续按品类扩展。

## 技术栈

Next.js + DeepSeek API + Supabase · Netlify 部署

## 已知局限（待验证）

- 店员真实日活留存率
- 多品类（保健品/医疗器械/处方药）内容泛化能力
- 商业化路径（SaaS 订阅 vs 连锁品牌采购）
